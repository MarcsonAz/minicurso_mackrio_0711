## Oficina de R - Mackenzie Rio - Ciências Econômicas - Relatórios com dados

Material para o minicurso de R palestrado na Mackenzie Rio em 07/11/2025



## Arquivo oficinaR.zip para usar no CLOUD



Baixar arquivo o oficinaR.zip e carregar no projeto. Com a sessão aberta fazer o upload de *oficinaR.zip* e o sistema irá descompactar os relatórios e as tabelas de apoio.

Verifique o diretório, algo como: 'cloud/project/'. Pode rodar com o botão Knit.
